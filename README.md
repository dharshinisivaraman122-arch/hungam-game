# 🎮 Hangman Pro Max

An advanced web-based Hangman game built using Flask, HTML, CSS, and JavaScript.

## 🚀 Features

* User Login System
* Multiple Difficulty Levels
* Word Categories
* Hint System
* Score Tracking
* Statistics Dashboard
* Leaderboard
* Achievements
* Timer
* Dark / Light Mode
* Responsive Design
* Session Management

## 🛠 Technologies Used

* Python
* Flask
* HTML5
* CSS3
* JavaScript

## ⚙️ Installation

Clone the repository:

```bash
git clone <repository-url>
```

Install dependencies:

```bash
pip install flask
```

Run the application:

```bash
python app.py
```

Open in browser:

```text
http://127.0.0.1:5000
```

## 🎯 Gameplay

1. Enter your username.
2. Select difficulty.
3. Guess letters to reveal the hidden word.
4. Use hints strategically.
5. Earn points and unlock achievements.
6. Compete on the leaderboard.

## 📊 Features Included

* Dynamic scoring system

* Timer-based gameplay
* Achievement badges
